Dataset Interpretation
  • APP winner companies are distinct procurement winner identifiers derived from APP data.
  • Joined companies are APP winner companies that matched QKB subjects by exact normalized NIPT.
  • The joined dataset is smaller than the full QKB registry layer because it only contains QKB companies that also appear as APP winner companies.
  • OpenCorporates financial enrichment is a secondary subset and should be interpreted as data availability for exploratory analysis.
  • Coverage values are based on the current local database state and may change after pipeline reruns.
  • Missing values should be interpreted in relation to source availability and normalization rules.
